== Description ==

Este não é apenas um plugin, ele representa a infância de milhares de pessoas que cresceram assistindo a arte de Roberto Gómez Bolaños, que nos divertiu e emocionou enquanto o assistíamos em seus grandes seriados Chaves e Chapolin.

Ao ativar este plugin, você verá frases do seriado Chaves no topo do seu painel administrativo, e poderá sempre relembrar dos bons momentos eternizados por este grande artista.
